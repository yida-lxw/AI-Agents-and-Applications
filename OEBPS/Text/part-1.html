第一部分 大语言模型入门

本书开篇为后续内容奠定基础，我们将深入探讨构建大语言模型（LLM）驱动应用的“为什么要构建”与“如何构建”—明确其擅长的领域、面临的局限，以及为什么 LangChain、LangGraph 和 LangSmith 等框架对于打造可靠、实用的真实系统至关重要。在此过程中，你将掌握三大核心架构模式：引擎、聊天机器人与智能体，并深入理解支撑它们运行的关键技术，包括提示词工程与检索增强生成（RAG）。

我们还将聚焦人工智能开发中最关键的日常技能之一：提示词的设计与执行。我们将深入探讨如何为不同类型的任务设计提示词，借助单样本（one-shot）与少样本（few-shot）示例优化提示词效果，并通过 LangChain 的提示词模板与 OpenAI API 实现全流程自动化。到本部分结束时，你将超越初步探索阶段，具备构建目标明确、稳定可靠的大语言模型应用的能力。
